## How to Run

1. **Clone the project**
   ```bash
   git clone [repository URL]
   cd ensan_test
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the app**
   ```bash
   flutter run
   ```
